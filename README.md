# 🔌 ESP32 Mini PCB Design (Altium)

## 📌 Overview
This project presents the design and development of a custom ESP32 Mini development board using Altium Designer. The project includes complete schematic capture, PCB layout design, custom component libraries, and manufacturing-ready outputs.

The board is designed as a compact embedded solution suitable for IoT and microcontroller-based applications.

---

## 🛠 Tools & Software
- Altium Designer
- ESP32 Module
- Custom Schematic Library (.SchLib)
- Custom PCB Footprint Library (.PcbLib)

---

## 🔧 Design Features
- Compact 4-layer PCB design
- Optimized ground plane
- Proper decoupling capacitor placement
- Clean power routing
- Organized component placement
- Manufacturing-ready PCB layout
